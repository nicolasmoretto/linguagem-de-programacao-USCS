Oque é HTML?

Quem quando começou a programar já sabia que HTML não é uma linguagem de programação que atire a primeira pedra!
Por mais incrível que pareça, HTML não é uma linguagem de programação, mas pera, seu próprio nome já diz: HiperText Markup Language (Linguagem de Marcação de “Hipertexto”.) O HTML recebe esse nome por ser um texto que possui a capacidade de ter atalhos de navegação sob algumas palavras, imagens e botões desenhados em sua página.

Como criar um arquivo em HTML?

Para criar uma página de hipertexto, basta criar um arquivo texto (Ex: MEUARQUIVO.txt), mas modificar a extensão do arquivo para “.html” (EX: MEUARQUIVO.html).

Oque são Tags?

É como chamamos tecnicamente os limitadores dos elementos de um documento em HTML.
Essas tags estão sempre entre os sinais “ < ” e “>” - no exemplo a baixo, a tag de abertura do elemento que abrange todo o nosso documento.

A maioria das tags exigem uma tag de fechamento - no caso ao lado, ao final do documento precisa existir a tag < /html> - O caractere “/” antes do nome da tag, indica que é uma tag de fechamento.
